# Shopping Cart Task - Rlogical

## _Technology Used_

- **Node Version:** v20.17.0
- **Language:** TypeScript
- **Frontend:** NextJS 15
- **Backend:** NodeJS + ExpressJS
- **Database:** MongoDB
- **Time Taken:** 6.5 hours

## _How to run project_

**For Frontend**

```sh
cd frontend
npm i
npm run dev
```

**For Backend**

- **Note:** Create **shopping-cart** database with **products** collection in mongodb compass

```sh
cd backend
npm i
npm run dev
```
